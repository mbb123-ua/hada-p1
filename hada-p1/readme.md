# \# Práctica 1 – Uso de Git

# 

# P1. ¿Qué crees que significa la opción -n en el comando revert?

# 

# La opción -n evita que git cree el commit automáticamente, permitiendo revisar los cambios antes de confirmarlos.

# 

# P2. ¿Qué cambios se han producido al ejecutar el comando reset?

# 

# El comando reset --hard mueve HEAD a un commit anterior y elimina los cambios del índice y del directorio de trabajo.

# 

# P3. ¿Qué diferencias observas entre revert y reset?

# 

# Revert crea un nuevo commit que deshace cambios sin modificar el historial.

# Reset modifica el historial y puede eliminar commits.

# 

# P4. ¿Qué diferencias observas entre merge y rebase?

# 

# Merge combina ramas manteniendo el historial completo.

# Rebase reescribe los commits para obtener un historial lineal.
